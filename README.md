# Light Following Robot 🚗💡

## 📌 Description
This project is an IoT-based Light Following Robot developed using Arduino. The robot detects light intensity and automatically moves toward the light source.

---

## 🛠️ Components Used
- Arduino Uno
- LDR Sensors (2)
- Motor Driver (L298N)
- DC Motors
- Battery

---

## ⚙️ Working Principle
The robot uses two LDR sensors to detect light intensity.

- If left sensor detects more light → robot turns left  
- If right sensor detects more light → robot turns right  
- If both sensors detect equal light → robot moves forward  

This allows the robot to follow the direction of the light source.
