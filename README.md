# Light Following Robot

## Description
This project is an IoT-based robot that follows a light source using LDR sensors and Arduino.

## Features
- Detects light intensity
- Moves towards light source
- Real-time decision making

## Components Used
- Arduino
- LDR Sensors
- Motor Driver (L298N)
- DC Motors

## Working
The robot uses two LDR sensors to detect light intensity. Based on sensor values, the Arduino controls motors to move in the direction of higher light intensity.
